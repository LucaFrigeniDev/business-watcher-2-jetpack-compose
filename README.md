# business-watcher-2-jetpack-compose
BusinessWatcher is a database where you can take note on the market (competitors, customers).

Using :
- Kotlin
- Jetpack Compose (Framework)
- Dagger Hilt (DI)
- Room (ORM)
- Google Maps (API)
